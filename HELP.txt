
Undocumented functions

-lvol - prints sampled points in fractional coordiantes, with accessibility and pocket/channel ID information


-gridGprojdataBohr history_1.lvol out.cube history_1.cssr       - saves a grid with counts of number of points in particular volumetric bin, based on .lvol file

-gridGprojdataBohr list_lvol out.cube history_1.cssr - save a grid with counts of number of frames that have a particular bin, please not that list_lvol is a file with a list of .lvol files
